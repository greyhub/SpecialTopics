# How to run/build
#### Install packages
```ruby
$ npm install
```
#### Set up database
>Step 1: Install xampp<br>
>Step 2: Start Apache and MySQL modules<br>
>Step 3: Add new account in phpmyadmin<br>
#### Setup environment
>Create a .env file, set up like:<br>
```javascript
PORT=8080
SALT=SALT
DB_HOST=127.0.0.1
DB_PORT=3306
DB_NAME=test
DB_USERNAME=test
DB_PASSWORD=123456
```
#### Run
```ruby
$ npm start
```
#### Build
```ruby
$ npm run build
```
